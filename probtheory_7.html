7 СЛУЧАЙНЫЕ ВЕЛИЧИНЫ
7.1 ВИДЫ СЛУЧАЙНЫХ ВЕЛИЧИН


Случайная величина – величина, которая в результате испытания принимает одно значение, заранее неизвестное и зависящее от случайных причин, которые заранее не могут быть учтены


Обозначение случайных величин: \(X\), \(Y\), ...
Значения, которые принимает случайная величина \(X\), обозначаются \(x_1\), \(x_2\), ...
Для случайной величины \(Y\): \(y_1\), \(y_2\), ...


Различают два вида случайных величин:
1) дискретные случайные величины
2) непрерывные случайные величины


Дискретная случайная величина – величина, которая принимает отдельные значения с определенными заданными вероятностями


Два типа дискретных случайных величин:
1) с конечным числом значений (например: число попаданий из 100 выстрелов – случайная величина со 101 значением, а именно: 0, 1, 2, ..., 100)
2) с бесконечным числом значений (например: число выстрелов до первого попадания – случайная величина со значениями 1, 2, 3, ...)


Непрерывная случайная величина – величина, которая может принимать любые значения из замкнутого или открытого интервала


Примеры непрерывной случайной величины: дальность полета снаряда, время работы прибора до поломки, курс доллара и т. п.


Закон распределения случайной величины – всякое соотношение, устанавливающее связь между возможными значениями случайной величины и соответствующими им вероятностями

7.2 ДИСКРЕТНЫЕ СЛУЧАЙНЫЕ ВЕЛИЧИНЫ


Для дискретной случайной величины закон распределения может быть задан в виде таблицы, графика или с помощью интегральной функции распределения


При табличном задании первая строка содержит возможные значения случайной величины, а вторая – их вероятности, причем \(\sum_{i=1}^{n}{p_i}=1\)


Такая таблица называется рядом распределения дискретной случайной величины


Пример
В урне находятся 8 шаров, из которых 5 белые и 3 черные
Из нее вынимают наудачу 3 шара
Составить ряд распределения случайной величины \(X\) – числа вынутых белых шаров
Решение
\(X\) может принимать следующие значения: \(x_1=0\), \(x_2=1\), \(x_3=2\), \(x_4=3\)
Найдем соответствующие этим значениям вероятности:

\(P(X=0)=\frac{C_5^0⋅C_3^3}{C_8^3}=\frac{1}{56}\)
\(P(X=1)=\frac{C_5^1⋅C_3^2}{C_8^3}=\frac{15}{56}\)
\(P(X=2)=\frac{C_5^2⋅C_3^1}{C_8^3}=\frac{30}{56}\)
\(P(X=3)=\frac{C_5^3⋅C_3^0}{C_8^3}=\frac{10}{56}\) или \(P(X=3)=1-\frac{1}{56}-\frac{15}{56}-\frac{30}{56}=\frac{10}{56}\)
Примечание
Мы можем вынуть 0, 1, 2 или 3 белых шара


При графическом задании дискретной случайной величины на оси абсцисс откладываются всевозможные значения случайной величины, а на оси ординат – вероятности этих значений


Ломаная, получаемая соединением этих точек, называется многоугольником распределения


Пример
На рисунке изображен многоугольник распределения дискретной случайной величины \(X\)
Определить значение вероятности при \(X=5\)
Решение

\(p=1-0.05-0.2-0.05-0.3=0.4\)



Интегральная функция распределения – функция \(F(x)\), определяющая вероятность того, что случайная величина \(X\) примет значение, меньшее, чем \(x\), т. е.:
\(F(x)=P(X < x)\)


Свойства интегральной функции распределения:
1) \(F(x)\) ограничена, т. е. \(0 ≤ F(x) ≤ 1\)
2) \(F(x)\) – неубывающая функция: если \(x_1 > x_2\), то \(F(x_1) ≥ F(x_2)\)
3) \(\lim_{x→∞}{F(x)}=1\), \(lim_{x→-∞}{F(x)}=0\)
4) \(F(x)\) непрерывна слева, т. е. \(lim_{x→x_0-0}{F(x)}=F(x_0)\)
Замечание
Интегральная функция распределения любой дискретной случайной величины есть разрывная ступенчатая функция, скачки которой происходят в точках, соответствующих возможным значениям случайной величины и равны вероятностям этих значений
Сумма всех скачков равна 1


Пример
Дискретная случайная величина задана рядом распределения
Составить интегральную функцию распределения и начертить ее график

Решение
\(F(x)=\left\{\begin{array}{c}0, x ≤ 2\\0.2, 2 < x ≤ 4\\0.3, 4 < x ≤ 7\\0.7, 7 < x ≤ 9\\1, x > 9\end{array}\right\}\)



Две случайные величины (СВ) называются независимыми, если закон распределения одной из них не меняется от того, какие возможные значения приняла другая величина


Если известны распределения независимых дискретных случайных величин \(X:\binom{x_i}{p_i}\) и \(Y:\binom{y_i}{q_i}\), где \(i = 1, 2, …, n\) и \(j = 1, 2, …, m\); \(n, m ∈ N\), то операции над ними определяются следующим образом:
1) умножение СВ на число: \(C⋅X:\binom{C⋅x_i}{p_i}\), \(C=const\)
2) возведение СВ в степень: \(X^m:\binom{x_i^m}{p_i}\)
3) сумма (разность) СВ: \(X±Y:\binom{x_i±y_j}{p_i⋅q_j}\)
4) произведение СВ: \(X⋅Y:\binom{x_i⋅y_j}{p_i⋅q_j}\)


Замечание
Если среди значений \(x_i^m\), \(x_i±y_j\), \(x_i⋅y_j\) появляются одинаковые, то соответствующие им столбцы надо объединить в один, приписав ему суммарную вероятность

7.3 НЕПРЕРЫВНЫЕ СЛУЧАЙНЫЕ ВЕЛИЧИНЫ


Непрерывные случайные величины задаются с помощью интегральной \(F(x)\) или дифференциальной \(f(x)\) функции распределения


Определение и свойства интегральной функции распределения аналогичны как для дискретных, так и для непрерывных случайных величин


График интегральной функции распределения непрерывной случайной величины представляет собой функцию, непрерывную во всех точках


Вероятность попадания непрерывной случайной величины в интервал \((a;b)\) равна приращению ее функции распределения на этом интервале, т. е.:
\(P(a < X < b)=F(b)-F(a)\)


Для непрерывной случайной величины \(X\) вероятность любого отдельного взятого значения равна 0, т. е. \(P(X=x_1)=0\), а вероятность попадания \(X\) в интервал \((a;b)\) не зависит от того, является ли этот интервал открытым или закрытым, т. е.:
\(P(a< X < b)=P(a ≤ X < b)=P(a < X ≤ b)=P(a ≤ X ≤ b)\)


Дифференциальной функцией распределения вероятностей (или плотностью вероятности) непрерывной случайной величины называется первая производная от интегральной функции распределения:
\(f(x)=F^ʹ(x)\)


Свойства плотности вероятности:
1) \(f(x) ≥ 0\)
2) \(\int_{-∞}^∞{f(x)dx}=1\)
3) вероятность попадания непрерывной случайной величины \(X\) в интервал \((a;b)\) равна:
\(P(a < X < b)=\int_a^b{f(x)dx}\)


График плотности вероятности называется кривой распределения


Кривая распределения лежит не ниже оси абсцисс
Полная площадь фигуры, ограниченной кривой распределения и осью абсцисс, равна 1


Зная плотность вероятности, можно найти интегральную функцию распределения:
\(F(x)=\int_{-∞}^x{f(x)dx}\)


Пример 1
Задана интегральная функция распределения непрерывной случайной величины:
\(F(x)=\left\{\begin{array}{c}0, x < 3\\C⋅(x-3)^2, 3 ≤ x ≤ 5\\1, x > 5\end{array}\right\}\)
Найти:
1) коэффициент \(C\)
2) \(P(2 ≤ x ≤ 4)\)
3) \(f(x)\)
Решение
1) \(\lim_{x→x_0-0}{F(x)=F(x_0)}\), т. е. \(\lim_{x→5-0}{⁡F(x)=F(5)}\) или \(C⋅(5-3)^2=1\), откуда \(C=0.25\)
2) \(P(2 ≤ x < 4)=F(4)-F(2)=0.25⋅(4-3)^2-0=0.25\)
3) \(f⋅(x)=F^ʹ⋅(x)\), тогда:
\(f(x)=\left\{\begin{array}{c}0, x < 3\\0.5⋅(x-3), 3 ≤ x ≤ 5,\\0, x > 5\end{array}\right\}\)
Примечание
Пункт 1:
\(F(x)\) непрерывна слева (свойство 4 интегральной функции распределения)
\(x_0\) – последняя точка, которая граничит между 2-м значением функции \((C⋅(x-3)^2, 3 ≤ x ≤ 5)\) и единицей
Пункт 2:
См. вероятность попадания непрерывной случайной величины в интервал \((a,b)\) пункта 7.3
\(F(4)\) входит в промежуток \(3 ≤ x ≤ 5\)
\(F(2)\) входит в промежуток \(x < 3\)
Пункт 3:
Находим производные от каждой функции распределения


Пример 2
При каком значении параметра \(C\) функция
\(f(x)=\left\{\begin{array}{c}0, x < 1\\\frac{C}{x^4}, x ≥ 1\end{array}\right\}\)
может быть полностью распределения некоторой непрерывной случайной величины?
Решение
По свойству 2 плотности вероятности: \(\int_{-∞}^∞{f(x)dx}=1\)
Тогда:
\(\int_{-∞}^∞{f(x)dx}=\int_{-∞}^1{0dx}+\int_1^∞{\frac{C}{x^4}dx}=C⋅\int_1^∞{x^{-4}dx}=C⋅\frac{x^{-3}}{-3}|_1^∞=-\frac{C}{3}⋅\left(\frac{1}{∞^3}^{→0}-\frac{1}{1^3}\right)=\frac{C}{3}\)
Из уравнения \(\frac{C}{3}=1\) находим \(C=3\)
Примечание
\(f(x)\) делится на два участка с разными значениями, тогда разобьем интервал \((-∞;1)\) и \([1;+∞)\)


Пример 3
Найти интегральную функцию распределения \(F(x)\), если плотность вероятности имеет вид:
\(f(x)=\left\{\begin{array}{c}0, x < 1\\\frac{3}{x^4}, x ≥ 1\end{array}\right\}\)
Решение
Если \(x < 1\), то \(F(x)=\int_{-∞}^x{0dx}=0\)
Если \(x ≥ 1\), то
\(F(x)=\int_{-∞}^1{0dx}+\int_1^x{\frac{3}{x^4}dx}=0+3⋅\frac{x^{-3}}{-3}|_1^x=-\frac{1}{x^3}|_1^x=-\left(\frac{1}{x^3}-\frac{1}{1^3}\right)=1-\frac{1}{x^3}\)
Тогда интегральная функция распределения вероятностей примет вид:
\(F(x)=\left\{\begin{array}{c}0, x < 1\\1-\frac{1}{x^3}, x ≥ 1\end{array}\right\}\)


Пример 4
Дана случайная величина с интегральной функцией распределения \(F(x)=c+b⋅\arctan⁡{x}\)
Найти:
1) постоянные \(b\) и \(c\)
2) плотность вероятности \(f(x)\)
Решение
1)
По свойству 3 интегральной функции распределения: \(lim_{x→∞}{F(x)}=1\) и \(lim_{x→-∞}{F(x)}=0\)
Тогда:
\(lim_{x→∞}{(c+b⋅\arctan{⁡x})}=c+b⋅\arctan{(∞)}=c+b⋅\frac{π}{2}\)
\(lim_{x→-∞}{(c+b⋅\arctan{⁡x})}=c+b⋅\arctan{(-∞)}=c+b⋅\left(-\frac{π}{2}\right)=c-b⋅\frac{π}{2}\)
Решая систему, находим постоянные \(b\) и \(c\):
\(\left\{\begin{array}{c}c+b⋅\frac{π}{2}=1\\c-b⋅\frac{π}{2}=0\end{array}\right\}⇔\left\{\begin{array}{c}c=\frac{1}{2}\\b=\frac{1}{π}\end{array}\right\}\)
2)
С учетом решения пункта 1, имеем: \(F(x)=\frac{1}{2}+\frac{1}{π}⋅\arctan{⁡x}\)
Тогда, поскольку \(f⋅(x)=F^ʹ(x)\), то
\(F^ʹ(x)=\left(\frac{1}{2}+\frac{1}{π}⋅\arctan{⁡x}\right)=\frac{1}{π}⋅\frac{1}{1+x^2}\)
Искомая плотность вероятности примет вид: \(f(x)=\frac{1}{π(1+x^2)}\)
