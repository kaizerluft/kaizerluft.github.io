1 C++ ITERATOR


Итератор – структура данных, которая используется для обращения к определенному элементу в контейнерах STL
Аналогичен индексу в скобках []


Подключение заголовочного файла iterator
#include <iterator>


Объявление и инициализация итератора
vector<int>::iterator It; – объявить итератор
vector<int>::const_iterator It; – объявить итератор без доступа к изменению значений
vector<int>::iterator It = V.begin(); – объявить итератор и направить его на начало вектора V


Итератор должен быть такого же типа как и контейнер, на который он ссылается


Итератор на vector Итератор на list, set, map
Только с идентичными итераторами, указывающие на одинаковый контейнер и тип
Применять арифметические операции
It += 5;
– смещение итератора на 5 ячеек вправо
It -= 5;
– смещение итератора на 5 ячеек влево
 Недоступно
Доступен advance(It, -5);
Применять операции сравнения
if (It1 == It2)
if (It1 >= It2)
 Недоступны операции <>
Использовать операцию разыменования, т. е. обращаться к значению элемента, на которое указывает итератор
cout << *It;
– вывести значение элемента с позицией It
*It += 5;
– прибавить 5 к значению элемента с позицией It
Использовать инкрмент и дикримент
It++;
It--;
Передавать переменной разницу итераторов
int Diff = It1 - It2;


auto – автоматическое определение типа переменной
auto It = V.begin();
– эквивалент записи vector<int>::iterator It = V.begin();
