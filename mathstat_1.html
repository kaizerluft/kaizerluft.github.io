1 ЭЛЕМЕНТЫ МАТЕМАТИЧЕСКОЙ СТАТИСТИКИ


Математическая статистика – раздел математики, изучающий математические методы сбора, систематизации, обработки и интерпретации результатов наблюдений с целью изучения статистических закономерностей


Предметом математической статистики является изучение случайных величин (или случайных событий, процессов) по результатам наблюдений


Целью математической статистики является создание методов сбора и обработки статистических данных для получения научных и практических выводов


Задачи математической статистики:
– указание методов сбора и группировки статистических сведений, полученных в результате экспериментов или наблюдений
– разработка методов анализа статистических данных






1 ЭЛЕМЕНТЫ МАТЕМАТИЧЕСКОЙ СТАТИСТИКИ
1.1 ВАРИАЦИОННЫЕ РЯДЫ


Генеральная совокупность – совокупность всех подлежащих изучению объектов или возможных результатов всех мыслемых наблюдений, производимых в неизменных условиях над одним объектом


Пусть из генеральной совокупности извлечена выборка объема \(n\), в которой значения \(x_1\) некоторого исследуемого признака \(X\) наблюдалось \(n_1\) раз, значения \(x_2\) – \(n_2\) раз, ..., значения \(x_m\) – \(n_m\) раз
Значения \(x_i\) – варианты, а \(n_i\) – их частоты
При этом \(\sum_{i=1}^m{n_i}=n\)


Вариационный ряд – ранжированный в порядке возрастания (или убывания) ряд вариант с соответствующими им частотами


Относительные частоты – отношения частот вариант к объему выборки:
\(ω_i=\frac{n_i}{n}\)


Сумма относительных частот равна 1, т. е. \(\sum_{i=1}^m{ω_i}=1\)


Статистический ряд – перечень вариант и соответствующих им частот (относительных частот)


Пример 1
Имеются результаты 20 измерений диаметра:
10.0; 10.1; 10.2; 10.1; 9.8; 9.9; 10.0; 10.0; 10.2; 10.0; 10.0; 9.9; 10.0; 10.1; 10.0; 9.9; 10.0; 10.1; 10.1; 10.0
Представить эти данные с помощью таблиц распределения по частотам и относительным частотам
Решение

Объем выборки \(n=1+3+9+5+2=20\)
Сумма всех частот равна объему выборки
Сумма всех относительных частот равна 1
Примечание
\(0.05=\frac{1}{20}\); \(0.15=\frac{3}{20}\)...


Пример 2
Из генеральной совокупности извлечена выборка объема \(n=200\)
Найти частоту варианты \(x_2=6\)

Решение
\(n_2=200-60-35-50=55\)


Пример 3
Статистическое распределение выборки имеет вид:

Найти значение относительной частоты \(ω_3\)
Решение
\(ω_3=1-0.45-0.15-0.1=0.3\)


Пример 4
Из генеральной совокупности извлечена выборка объема \(n=100\)
Найти относительную частоту варианты \(x_4=4\)

Решение
Частота \(n_4=100-15-20-25=40\), тогда относительная частота \(ω_4=\frac{n_4}{n}=\frac{40}{100}=0.4\)


Пример 5
Из генеральной совокупности извлечена выборка объема \(n=400\) Найти частоту варианты \(x_1=4\)

Решение
Относительная частота \(ω_1=1-0.25-0.15-0.4=0.2\), тогда частота \(n_1=ω_1*n=0.2*400=80\)


Дискретный вариационный ряд – ряд, в котором любые его варианты отличаются на постоянную величину


Непрерывный (интеральный) вариационный ряд – ряд, в котором варианты могут отличаться один от другого на сколь угодно малую величину


Преобразование интервального ряда в дискретный
Для преобразования интервального статистического ряда в дискретный следует интервалы заменить соответствующими им серединами, а частоты взять из интервального ряда


Преобразование дискретного ряда в интервальный
Для преобразования дискретного статистического ряда в интервальный необходимо:
1) установить величину интервалов
2) получить шкалу интервалов
3) в соответствии с этой шкалой сгруппировать данные


1) Для определение величины интервала используется формула Стерджесса:
\(h=\frac{x_{max}-x_{min}}{1+3.322*\lg{n}}\)
где
\(x_{max}\), \(x_{min}\) – наибольшее и наименьшее значения признака
\(1+3.322*\lg{n}=1+\log_2{n}\) – число интервалов
2) Шкала интервалов формируется следующим образом:
\(a_1=x_{min}\)
\(a_2=a_1+h\)
\(a_3=a_2+h\)
...
3) Границы интервалов формируются до тех пор, пока не превысят \(x_{max}\): \([a_1;a_2)\), \([a_2;a_3)\), ..., \([a_{m-1};a_m]\), где \(a_m ≥ x_{max}\)
Во второй строке статистического ряда записывается количество наблюдений, попавших в каждый интервал


Пример 6
Имеются данные о стаже рабочих цеха: 6, 6, 10, 10, 7, 2, 2, 5, 8, 8, 12, 9, 10, 10, 7, 7, 6, 7, 2, 3
Построить:
1) дискретный вариационный ряд
2) интервальный вариационный ряд
Решение
1) построение дискретного вариационного ряда
Находится объем выборки, т. е. \(n\) – общее количество чисел (\(n=20\))
Составляется таблица, где \(x_i\) – варианты (числа из условия), расположенные в порядке возрастания, \(n_i\) – сколько раз встречается каждое число

2) построение интервального вариационного ряда
2.1) находится величина интервала:
\(h=\frac{x_{max}-x_{min}}{1+3.322*\lg{n}}=\frac{12-2}{1+3.322*\lg{20}}=\frac{10}{5.3}=1.9\)
2.2) Шкала интервалов формируется так:
\(a_1=x_{imin}=2\)
\(a_2=a_1+h=2+1.9=3.9\)
\(a_3=a_2+h=3.9+1.9=5.8\)
\(a_4=7.7\)
\(a_5=9.6\)
\(a_6=11.5\)
\(a_7=13.4 > x_{max}=12\)
2.3) составляется таблица, в которой в первой строке формируются границы интервалов, а число во второй строке \(n_{i}\) – общая сумма частоты встреч всех чисел дискретного ряда, попадающих в соответствующий интервал

1.2 ЭМПИРИЧЕСКАЯ ФУНКЦИЯ РАСПРЕДЕЛЕНИЯ


Эмпирическая функци распределения – функция \(F^*(x)\), определяющая для каждого значения \(x\) относительную частоту того, что признак \(X\) примет значение, меньше заданного, т. е. относительную частоту события \(X < x\):
\(F^*(x)=\frac{n_x}{n}\)
где
\(n_x\) – число наблюдений, меньших \(x\)
\(n\) – объем выборки


Свойства эмпирической функции распределения:
1) \(F^*(x)\) ограничена: \(0 ≤ F^*(x) ≤ 1\)
2) \(F^*(x)\) – неубывающая функция: если \(x_1 > x_2\), то \(F^*(x_1) ≥ F^*(x_2)\)
3) \(\lim_{x→∞}{F^*(x)}=1\), \(\lim_{x→-∞}{F^*(x)=0}\)
4) \(F^*(x)\) непрерывна слева, т. е. \(\lim_{x→x_0-0}{F^*(x)}=F^*(x_0)\)


Эмпирическая функция распределения \(F^*(x)\) служит оценкой теоретической функции распределения \(F(x)\) генеральной совокупности


При больших \(n\) вероятность отличия этих функций друг от друга близка к 1:
\(\lim_{x→∞}{P\left\{|F(x)-F^*(x)| < ε\right\}}=1\), \(ε > 0\)


Пример 7
Построить эмпирическую функцию по заданному распределению выборки

Решение
Объем выборки равен \(n=1+3+2+4=10\)
Найдем значения относительных частот \(ω_i\) выборки

Искомая эмпирическая функция примет вид:
\(F^*(x)=\left\{\begin{array}{c}0, x ≤ 2\\0.1, 2 < x ≤ 5\\0.4, 5 < x ≤ 8\\0.6, 8 < x ≤ 11\\1, x > 11\end{array}\right\}\)

1.3 ГРАФИЧЕСКОЕ ИЗОБРАЖЕНИЕ ВАРИАЦИОННЫХ РЯДОВ


Для изображения вариационных рядов обычно используются полигон, гистограмма, кумулятивная кривая и огива


Дискретный вариационный ряд графически изображается полигоном
Каждую пару значений \((x_i;n_i)\) из распределения выборки можно трактовать как точку на координатной плоскости
Точно так же можно рассматривать и пары значений \((x_i;ω_i)\) относительного распределения выборки


Полигон частот – ломанная, отрезки которой соединяют точки \(x_i;n_i\)


Полигон относительных частот – ломанная, отрезки которой соединяют точки \(x_i;ω_i\)


Пример 8
Построить полигон частот для дискретного вариационного ряда по данным примера 6

Интервальный вариационный ряд графически изображается гистограммой


Гистограмма частот – ступенчатая фигура, состоящая из прямоугольников, основаниями которых служат частичные интервалы длины \(h\), а высоты равны отношению \(\frac{n_i}{h}\) (плотность частоты)


Геометрический смысл гистограммы частот: площадь гистограммы частот равна сумме всех частот, т. е. объему выборки \(n\)

Геометрический смысл гистограммы относительных частот: площадь гистограммы относительных частот равна сумме всех частот, т. е. 1

Кумулятивная кривая (кумулята) строится как для дискретных, так и для интервальных вариационных рядов

где
\(n_i^{нак}\) – накопленная частота
\(ω_i^{нак}=\frac{n_i^{нак}}{n}\)

Верхним границам интервалов соответствуют накопленные частоты, а нижней границе первого интервала – накопленная частота, равная 0

Разновидностью кумуляты является кривая концентрации или график Лоренца


Для построения кривой концентрации на обе оси прямоугольной системы координат наносится масштабная шкала в процентах от 0 до 100
При этом на оси абсцисс указывают накопленные частоты, а на оси ординат – накопленные значения доли (в процентах) по объему признака

1.4 ТОЧЕЧНЫЕ ОЦЕНКИ ПАРАМЕТРОВ РАСПРЕДЕЛЕНИЯ
1.4.1 ПОНЯТИЕ ОЦЕНКИ ПАРАМЕТРОВ


Параметры – числовые характеристики всей генеральной совокупности


Т. к. всю генеральную совокупность изучить достаточно часто не представляется возможным, о параметрах судят по выборочным характеристикам
На основании выборочных данных можно получить лишь приближенное значение параметра, которое является его оценкой
Пусть \(\tilde{Θ}_n\) – точечная оценка для параметра \(Θ\) генеральной совокупности
Многократные выборки одинакового объема дадут различные значения параметра \(Θ\)
В этом случае возникает проблема выбора наилучшей оценки


Чтобы выбранная оценка была наилучшей, она должна удовлетворять свойствам несмещенности, эффективности и состоятельности:
1. Оценка \(\tilde{Θ}_n\) несмещенная, если ее математическое ожидание равно оцениваемому параметру: \(M\left\lceil\tilde{Θ}_n\right\rceil\)
2. \(\tilde{Θ}_n\) эффективная, если она имеет наменьшую дисперсию при заданном объеме выборки
3. Оценка состоятельная, если при \(n→∞\) стремится по вероятности к оцениваемому параметру:
\(\lim_{n→∞}{P\left(\left|\tilde{Θ}_n-Θ\right| < ε\right)}=1\)

1.4 ТОЧЕЧНЫЕ ОЦЕНКИ ПАРАМЕТРОВ РАСПРЕДЕЛЕНИЯ
1.4.2 ЧИСЛОВЫЕ ХАРАКТЕРИСТИКИ ВЫБОРКИ


Репрезентативная выборка – выборка, в которой все основные признаки генеральной совокупности, из которой извлечена данная выборка, представлены приблизительно в той же пропорции или с той же частотой, с которой данный признак выступает в этой генеральной совокупности
При достаточно большом значении объема выборки нет разницы в выборе смещенной или несмещенной оценки параметра, при малом \(n\) необходимо рассматривать несмещенную оценку, поскольку выборка будет репрезентативной, т. е. полностью представлять генеральную совокупность
Для выборки можно определить ряд числовых характеристик: выборочное среднее, выборочная дисперсия, выборочное среднеквадратическое отклонение, размах вариации, мода, медиана


Пусть статистическое распределение выборки имеет вид:

Выборочное среднее (обозначается \(\bar{x}\)) – среднее арифметическое всех значений выборки:
\(\bar{x}=\frac{1}{n}*\sum_{i=1}^m{x_i*n_i}\)
или
\(\bar{x}=\sum_{i=1}^m{x_i*ω_i}\)
Выборочное среднее показывает среднее значение, вокруг которого группируются варианты \(\bar{x}\) является несмещенной и состоятельной оценкой математического ожидания


Пример 1
Из генеральной совокупности извлечена выборка
Найти несмещенную оценку математического ожидания

Решение
Найдем объем выборки: \(n=2+7+1=10\)
Выборочное среднее: \(\bar{x}=\frac{3*2+5*7+9*1}{10}=5\)


Пример 2
Проведено 4 измерения (без систематических ошибок) некоторой случайной величины (в мм): 5, 8, 6, 9
Найти несмещенную оценку математического ожидания
Решение

Выборочное среднее \(\bar{x}=\frac{5*1+6*1+8*1+9*1}{4}=7\)


Выборочная дисперсия – среднее арифметическое квадратов отклонений значений выборки от выборочной средней:
\(S^2=\frac{1}{n}\sum_{i=1}^m{\left(x_i-\bar{x}\right)^2*n_i}\)


Выборочная дисперсия показывает меру разброса данных вокруг среднего значения
Она является состоятельной и смещенной оценкой дисперсии


Алгоритм расчета выборочной дисперсии:
1. Вычисляется объем выборки \(n\)
2. Находится выборочная средняя \(\bar{x}\)
3. Определяется величина \(\bar{x^2}=\frac{1}{n}\sum_{i=1}^m{x_i^2*n_i}\)
4. Выборочная дисперсия равна \(S^2=\bar{x^2}-\left(\bar{x}\right)^2\)


Пример 3
Из генеральной совокупности извлечена выборка
Найти выборочную дисперсию

Решение
1. \(n=2+7+1=10\)
2. \(\bar{x}=\frac{3*2+5*7+9*1}{10}=5\)
3. \(\bar{x^2}=\frac{3^2*2+5^2*7+9^2*1}{10}=\frac{18+175+81}{10}=27.4\)
4. \(S^2=27.4-5^2=2.4\)


Выборочная дисперсия является смещенной оценкой генеральной дисперсии, т. е. математическое ожидание выборочной дисперсии не равно оцениваемой генеральной дисперсии, а равно:
\(M\left[S^2\right]=\frac{n-1}{n}D\)


Для исправления выборочной дисперсии достаточно умножить ее на дробь \(\frac{n}{n-1}\), называемую поправкой Бесселя (при больших объемах она мало отличается от 1)
При этом получается исправленная дисперсия \(S^{*2}\), которая является несмещенной оценкой генеральной дисперсии:
\(S^{*2}=\frac{n}{n-1}*S^2\)


Пример 4
В результате измерений некоторой физической величины одним прибором (без систематических ошибок), получены следующие результаты (в мм): 6, 9, 7
Найти несмещенную оценку дисперсии
Решение

1. \(n=1+1+1=3\)
2. \(\bar{x}=\frac{6*1+7*1+9*1}{3}=\frac{22}{3}\)
3. \(\bar{x^2}=\frac{6^2*1+7^2*1+9^2*1}{3}=\frac{36+49+81}{3}=\frac{166}{3}\)
4. \(S^2=\frac{166}{3}-\left(\frac{22}{3}\right)^2=\frac{14}{9}\)
5. \(S^{*2}=\frac{3}{3-1}*\frac{14}{9}=\frac{7}{3}\)


Для оценки степени отклонения от среднего значения удобно иметь дело с величиной той же размерности, что и величина \(X\)
Для этого вводится понятие выборочного среднеквадратического отклонения


Выборочное среднеквадратическое отклонение – арифметическое значение корня квадртаного из его дисперсии:
\(S=\sqrt{S^2}\)


Тогда исправленное выборочное среднеквадратическое отклонение равно:
\(S^*=\sqrt{S^{*2}}\)


Размах вариации (обозначается \(R\)) – разность между максимальной и минимальной вариантами или длина интервала, которому принадлежат все варианты выборки:
\(R=x_{max}-x_{min}\)


Размах показывает величину разброса значений случайной величины в выборке


Пример 5
Найти размах вариационного ряда: 5, 7, 8, 10, 10, 14
Решение
Размах равен \(R=14-5=9\)


Мода (обозначается \(\tilde{M}_0\)) – значение варианты, имеющей наибольшую частоту


Особенность моды как меры центральной тенденции заключается в том, что она не изменяется при изменении крайних членов ряда, т. е. обладает определенной устойчивостью к вариации признака


Пример 6
Найти моду для выборки
Решение

\(n_{imax}=10\), следовательно \(\bar{M}_0=3\)


Медиана (обозначается \(\tilde{M}e\)) – значение признака, относительно которого вариационный ряд делится на две одинаковые части по числу вариант
Если ряд имеет четное число членов, т. е. \(m=2k\), то:
\(\tilde{M}e=\frac{x_k+x_{k+1}}{2}\)


Пример 7

Решение

Медианой является среднее арифметическое выделенных чисел:
\(\tilde{M}e=\frac{8+10}{2}=9\)

Решение

\(x_{26}=x_{27}=...=x_{25+12}=x_{37}=4\)

Тогда медиана равна \(\tilde{M}e=\frac{x_{30}+x_{31}}{2}=\frac{4+4}{2}=4\)


Если ряд имеет нечетное число членов, т. е. \(m=2k+1\), то:
\(\tilde{M}e=x_{k+1}\)


Пример 9
Найти медиану вариационного ряда: 3, 4, 5, 9, 10, 10, 12, 12, 17

1 ЭЛЕМЕНТЫ МАТЕМАТИЧЕСКОЙ СТАТИСТИКИ
1.5 ИНТЕРВАЛЬНЫЕ ОЦЕНКИ ПАРАМЕТРОВ РАСПРЕДЕЛЕНИЯ
1.5.1 ОСНОВНЫЕ ПОНЯТИЯ


В ряде задач требуется найти для неизвестного параметра не только подходящее значение, но и оценить его точность и надежность


Доверительный интервал для параметра \(Θ\) – интервал \((ε_1;ε_2)\), в котором с заранее заданной вероятностью содержится истинное значение этого парметра:
\(P(ε_1 < Θ < ε_2)=γ\)
где \(γ\) – доверительная вероятность или надежность


Пусть для параметра \(Θ\) на основании выборочных данных получена несмещенная оценка \(\tilde{Θ}\)
Найдем такое положительное значение \(ε\), для которого событие \(\left|Θ-\tilde{Θ}\right| < ε\) с вероятностью \(γ\) можно считать достоверным
\(\left|Θ-\tilde{Θ}\right| < ε\)
\(-ε < Θ-\tilde{Θ} < ε\)
\(\tilde{Θ}-ε < Θ < \tilde{Θ}+ε\)


Доверительный интервал: \(\left(\tilde{Θ}-ε;\tilde{Θ}+ε\right)\)
\(ε\) – погрешность доверительного интервала, \(ε < \tilde{Θ}\)


Для доверительного интервала вида \((a;b)\) справедливы следующие соотношения:
\(ε=\frac{b-a}{2}\)
\(\tilde{Θ}=\frac{a+b}{2}\)


Пример 1
Дан доверительный интервал (15.5; 15.9) для оценки математического ожидания нормально распределенного количественного признака
Найти:
1) точность этой оценки
2) точечную оценку математического ожидания
Решение
1) \(ε=\frac{15.9-15.5}{2}=\frac{0.4}{2}=0.2\)
2) \(\tilde{Θ}=\frac{15.5+15.9}{2}=\frac{31.4}{2}=15.7\)


Пример 2
Точечная оценка математического ожидания нормально распределенного количественного признака равна \(17.8\)
Тогда его интервальная оценка может иметь вид:
1) (13.5; 20)
2) (16; 18)
3) (17.3; 18.3)
Решение
1) \(\left\{\begin{array}{c}17.8-ε=13.5\\17.8+ε=20\end{array}\right\}⇔\left\{\begin{array}{c}ε=4.3\\ε=2.2\end{array}\right\}\) – не верно
2) \(\left\{\begin{array}{c}17.8-ε=16\\17.8+ε=18\end{array}\right\}⇔\left\{\begin{array}{c}ε=1.8\\ε=0.2\end{array}\right\}\) – не верно
3) \(\left\{\begin{array}{c}17.8-ε=17.3\\17.8+ε=18.3\end{array}\right\}⇔\left\{\begin{array}{c}ε=0.5\\ε=0.5\end{array}\right\}\) – верно
Ответ: 3






1 ЭЛЕМЕНТЫ МАТЕМАТИЧЕСКОЙ СТАТИСТИКИ
1.5 ИНТЕРВАЛЬНЫЕ ОЦЕНКИ ПАРАМЕТРОВ РАСПРЕДЕЛЕНИЯ
1.5.2 ДОВЕРИТЕЛЬНЫЙ ИНТЕРВАЛ ДЛЯ ОЦЕНКИ МАТЕМАТИЧЕСКОГО ОЖИДАНИЯ НОРМАЛЬНО РАСПРЕДЕЛЕННОЙ СЛУЧАЙНОЙ ВЕЛИЧИНЫ ПРИ ИЗВЕСТНОМ СРЕДНЕКВАДРАТИЧЕСКОМ ОТКЛОНЕНИИ


Если исследуемая случайная величина распределена по нормальному закону с известым среднеквадратическим отклонением \(σ\), то доверительный интервал для оценки математического ожидания определяется неравенством:
\(\bar{x}-\frac{tσ}{\sqrt{n}} < a < \bar{x}+\frac{tσ}{\sqrt{n}}\)
где:
\(\tilde{Θ}=\tilde{x}\) – точечная оценка математичесокго ожидания (выборочное среднее)
\(ε=\frac{tσ}{\sqrt{n}}\) – точность оценки
\(n\) – объем выборки
\(t\) – квантиль нормального распределения или значение аргумента функции Лапаласа, при котором \(Ф(t)=\frac{γ}{2}\)


Замечания:
1) если \(x > 5\), то \(Ф(x)=0.5\)
2) функция Лапласа нечетная, т. е. \(Ф(-x)=-Ф(x)\)


Пример
Найти доверительный интервал для оценки математического ожидания \(a\) нормального распределения с надежностью \(γ=0.95\), зная:
\(\bar{x}=2.3\) – выборочное среднее
\(n=49\) – объем выборки
\(σ=1.4\) – генеральное среднеквадратическое отклонение
Решение
Воспользуемся формулой \(Ф(t)=\frac{γ}{2}=\frac{0.95}{2}=0.475\)
Далее по таблице находим \(t=1.96\)
Искомый доверительный интервал:
\(2.3-\frac{1.96*1.4}{\sqrt{49}} < a < 2.3+\frac{1.96*1.4}{\sqrt{49}}\)
\(1.908 < a < 2.692\)






1 ЭЛЕМЕНТЫ МАТЕМАТИЧЕСКОЙ СТАТИСТИКИ
1.5 ИНТЕРВАЛЬНЫЕ ОЦЕНКИ ПАРАМЕТРОВ РАСПРЕДЕЛЕНИЯ
1.5.3 ДОВЕРИТЕЛЬНЫЙ ИНТЕРВАЛ ДЛЯ ОЦЕНКИ МАТЕМАТИЧЕСКОГО ОЖИДАНИЯ НОРМАЛЬНО РАСПРЕДЕЛЕННОЙ СЛУЧАЙНОЙ ВЕЛИЧИНЫ ПРИ НЕИЗВЕСТНОМ СРЕДНЕКВАДРАТИЧЕСКОМ ОТКЛОНЕНИИ


Если среднеквадратическое отклонение нормально распределенной случайной величины неизвестно, то доверительный интервал для оценки математического ожидания определяется соотношением:
\(\bar{x}-\frac{t_γ*S^*}{\sqrt{n}} < a < \bar{x}+\frac{t_γ*S^*}{\sqrt{n}}\)
где
\(\tilde{Θ}=\bar{x}\) – точечная оценка математического ожидания
\(ε=\frac{t_y*S^*}{\sqrt{n}}\) – точность оценки
\(S^*\) – исправленное выборочное среднеквадратическое отклонение
\(n\) – объем выборки
\(t_γ\) – квантиль распределения Стьюдента


Квантиль распределения Стьюдента определяется:
1) по таблице в зависимости от объема \(n\) и надежности \(γ\)
2) по таблице в зависимости от числа степеней свободы \(k=n-1\) и уровня значимости \(α=1-γ\)


Пример
Найти доверительный интервал для оценки математического ожидания \(a\) нормального распределения надежностью \(γ=0.95\), зная:
\(\bar{x}=8\) – выборочное среднее
\(n=10\) – объем выборки
\(S^{*2}=5.76\) – исправленная выборочная дисперсия
Решение
Найдем исправленное среднеквадратическое отклонение:
\(S^*=\sqrt{5.76}=2.4\)
Квантиль распределения Стьюдента определим двумя способами:
1) \(n=10\), \(γ=0.95\), по таблице найдем \(t_γ=2.26\)
2) \(k=10-1=9\), \(α=1-0.95=0.05\), по таблице найдем \(t_γ=2.26\)
Искомый доверительный интервал: \(8-\frac{2.26*2.4}{\sqrt{10}} < a < 8+\frac{2.26*2.4}{\sqrt{10}}\)
\(6.3 < a < 9.7\)
